In this folder you can find the following files:
-student.h: the interface to manage students.
-studentMananger.h: the interface of students' managers.
-tester.c: a program that can be used to test your implementations.
-files testN.txt : is a file that can be used to test your code. After you compiled your code 
		   and obtained an executable exec. Then you can run ./exec<testN.txt. the result
		   of this should be the same as what's in outN.txt. 
-files outN.txt : see previous entry.
